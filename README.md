## Gameplay
### Basic Controls
- Move: ``WASD`` OR ``ARROW KEYS``
- Jump: ``SPACE``
- Shrink: ``SCROLL UP`` OR ``G``
- Grow: ``SCROLL DOWN`` OR ``H``
- Change Perspective: ``C``
